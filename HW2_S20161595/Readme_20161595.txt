#처음 프로그램을 실행하면 켜지는 [500, 500]크기의 창에서의 기능

기능1 : 마우스를 클릭할 때의 마우스의 위치(x, y)부터 화면의 중앙(250, 250)위치까지 선이 다시 그려져 화면에 나타난다.
(기능1 사용법 : ctrl키를 누른 상태로, 마우스 왼쪽버튼을 윈도우창 안에서 클릭한다.)

기능2 : 현재 화면에 그려져있는 선의 시작점과, 끝점의 좌표가 콘솔창에 출력된다.
(기능2 사용법 : shift키를 누른 상태로, 키보드 f(F)키를 누른다.)

기능3 : x좌표 250(윈도우 창의 중앙)을 기준으로 오른쪽에서 마우스를 움직이면 화면에 그려져 있는 선이
         점점 하얘지고, x좌표 250(윈도우 창 중앙)을 기준으로 왼쪽에서 움직이면 화면에 그려져 있는 선이 점점 까맣게 된다.
(기능3 사용법 : alt키와 마우스 왼쪽버튼을 누른 상태로 윈도우창의 중앙을 기준으로 오른쪽 또는 왼쪽 부분에서 마우스를 움직인다.)
